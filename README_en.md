# CloudDrive RPC 

`> cd rpc`

#### Introduction

Building RPC applications based on cloud drive services. 

#### Run the example program 

```
cargo run --bin rpc -- \
    --username <username> \
    --password <password> \
    --uri <webdav_uri> \
    --domain <domain> \
    --workdir <workdir> \
    --nodeid <node_id>
```
